# Joseph Nganga Portfolio

A professional static portfolio website for Joseph Nganga, an Information Technology student at Kabarak University. The site presents a personal profile, technical skills, selected projects, education background, and contact links in a clean responsive layout.

## Features

- Responsive single-page portfolio layout
- Light and dark theme toggle with saved preference
- Animated hero name typing effect
- Mobile-friendly navigation menu
- Active section highlighting while scrolling
- Reveal-on-scroll card animations
- Contact links for email, phone, LinkedIn, and GitHub

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Getting Started

No build step is required. Open `index.html` directly in a browser, or run a local static server from the project folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customization

- Edit portfolio content in `index.html`
- Update visual styling, colors, spacing, and responsive rules in `assets/css/styles.css`
- Adjust interactive behavior in `assets/js/script.js`
- Replace or update `assets/icons/favicon.svg` for branding changes
